# Habit Tracker

Webová appka pro sledování denních, týdenních a měsíčních zvyků. Každý zvyk lze sdílet přes unikátní read-only URL — bez nutnosti registrace pro diváka.

## Stack
Next.js 15 + Supabase + Tailwind CSS + Vercel

## Lokální vývoj

```bash
npm install
npm run dev
```

Otevři [http://localhost:3000](http://localhost:3000).

Potřebuješ `.env.local` s Supabase credentials (viz `.env.example`).
